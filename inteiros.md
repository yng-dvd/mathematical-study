### 1 - Simétricos

  Quando falamos do conjunto dos números inteiros, a grande diferença nas suas propriedades em relação ao conjunto dos números naturais se dá principalmente na implementação dos conceitos dos simétricos. Se no conjunto dos números naturais nós tínhamos uma pequena aplicação subliminar desse conceito, no conjunto dos inteiros ele se dará por completo. Chamamos de simétrico de a um número -a tal que a + (-a) = 0. A partir disso podemos traçar até mesmo o conceito de divisão que estabelecemos ainda no nosso tópico dos números naturais, ainda que mesmo no conjunto dos inteiros tenhamos limitações. Conferir: [multiplicação.](https://github.com/yng-dvd/mathematical-study/blob/main/naturais.md#33-multiplica%C3%A7%C3%A3o)

A multiplicação continua tendo uma relação com a ordem, onde a < b, c > 0: a x c < b x c, no entanto, quando  a < b e com c < 0: a x c > b x c. Exemplo:

1 < 7
Supondo que 1 x (-2) < 7 x (-2) = -2 < -14. Não confere, visto que -2 > -14.

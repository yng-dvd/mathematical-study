## Problema 2.5
Quantos algarismos são usados para numerar um livro de 300 páginas? Quantas vezes usa-se cada algarismo?

### Resolução
[1,9] = 9 números, sendo 1 o número de algarismos de cada número: 9 x 1 = 9;
[10, 99] = 90 números, sendo 2 o número de algarismos de cada número: 90 x 2 = 180
[100, 300] = 201 números, sendo 3 o número de algarismos de cada número: 201 x 3 = 603

603 + 180 + 9 = 792 algarismos.

[1, 9]: 1 vez
[10, 99]: 19 vezes e 9 vezes para o 0.
[100, 300]: 0 = 20 + 20 + 2 = 42 vezes; 1 = 140 vezes; 2 = 140 vezes; 3 = 41 vezes; 4-9 = 40 * 6 = 240 vezes.

Para o 0: 51 vezes;
Para o 1: 160 vezes;
Para o 2: 160 vezes;
Para o 3: 61 vezes;
Para o 4-9: 360 vezes;

Total: 792 vezes

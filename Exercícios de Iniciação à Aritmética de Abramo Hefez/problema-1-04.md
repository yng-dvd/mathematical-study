### Problema 1.4

Mostre que dados três números naturais $a$, $b$ e $c$, quaisquer, se $a + c = b + c$, então $a = b$.

#### Resposta

Sendo $c = 0$:
$$a + 0 = a$$
$$b + 0 = b$$
$$a + 0 = b + 0 \implies a = b$$

Sendo $c = n$:
$$a + n = b + n \implies a = b$$

Sendo $c = n + 1$:
$$a + (n + 1) = b + (n + 1)$$
$$(a + n) + 1 = (b + n) + 1$$
$$(a + n) = (b + n) \implies a = b$$

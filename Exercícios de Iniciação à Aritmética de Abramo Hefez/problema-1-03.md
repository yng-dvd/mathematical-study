### Problema 1.3

Utilizando as propriedades comutativa e associativa da adição, mostre que os 12 modos de somar três números $a$, $b$ e $c$:
$(a + b) + c$, $a + (b + c)$, $(a + c) + b$, $a + (c + b)$, $(b + a) + c$, $b + (a + c)$, $(b + c) + a$, $b + (c + a)$, $c + (b + a)$, $(c + a) + b$, $c + (a + b)$, $(c + b) + a$, dão o mesmo resultado.

#### Resposta
* $(a + b) + c$
* $a + (b + c) = (a + b) + c$ pela associativa; 
* $(a + c) + b = a + (c + b)$ pela associativa $= a + (b + c)$ pela comutativa $= (a + b) + c$ pela associativa; 
* $a + (c + b) = a + (b + c)$ pela comutativa $= (a + b) + c$ pela associativa; 
* $(b + a) + c = (a + b) + c$ pela comutativa; 
* $b + (a + c) = (b + a) + c$ pela associativa $= (a + b) + c$ pela comutativa; 
* $(b + c) + a = (c + b) + a$ pela comutativa $= c + (b + a)$ pela associativa $= (b + a) + c$ pela comutativa $= (a + b) + c$ pela comutativa;
* $b + (c + a) = b + (a + c)$ pela comutativa $= (b + a) + c$ pela associativa $= (a + b) + c$ pela comutativa; 
* $c + (b + a) = (b + a) + c$ pela comutativa $= (a + b) + c$ pela comutativa; 
* $(c + a) + b = c + (a + b)$ pela associativa $= (a + b) + c$ pela comutativa; 
* $c + (a + b) = (a + b) + c$ pela comutativa; 
* $(c + b) + a = c + (b + a)$ pela associativa $= (b + a) + c$ pela comutativa $= (a + b) + c$ pela comutativa.
